🎨 PaintScript

用绘画的方式编程，用编程的方式绘画

🎯 核心概念

画布 (Canvas): 程序的执行环境

画笔 (Brush): 数据处理工具

颜料 (Paint): 变量和数据

调色板 (Palette): 函数库

素描 (Sketch): 算法框架

混色 (Blend): 数据合并操作

🖌️ 基础语法

// 创建画布 画布 "我的作品" 宽度:800 高度:600 // 声明颜料(变量) 颜料 红色 = "#FF0000" 颜料 数量 = 10 // 选择画笔 画笔 细笔 尺寸:2 // 开始绘画(执行) 开始绘画 { 移动到 (100, 100) 画线到 (200, 200) }

🎨 控制结构

// 重复画法(循环) 重复画 5 次 { 画圆 半径:10 移动 右:50 } // 条件画法 如果 颜料不足 { 补充颜料 红色 } 否则 { 继续画 } // 分层画法(函数) 画法 画花朵(位置, 颜色) { 移动到 位置 选择颜色 颜色 画圆 半径:20 }

🌈 调色板(内置函数)

// 颜色调色板 混色(红色, 蓝色) → 紫色 渐变(起始色, 结束色, 步数) 随机色() // 几何调色板 画直线(起点, 终点) 画曲线(控制点列表) 画多边形(顶点列表) // 艺术调色板 模糊效果(强度) 水彩效果(透明度) 油画效果(笔触大小)
